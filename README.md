# NFC Tracker

App de Flutter para trackear hidratación y entrenamientos mediante NFC stickers.

## 🎯 Características
- 💧 Track de agua bebida con NFC
- 💪 Registro de entrenamientos
- 📱 Notificaciones locales
- 📊 Historial de sesiones

## 🚀 Instalación
1. Instala Flutter
2. `git clone ...`
3. `flutter pub get`
4. `flutter run`

## 🔧 Requisitos
- Android 10.0+
- NFC habilitado
